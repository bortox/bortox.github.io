Siccome non vogliamo che lo script di Mathjax venga caricato - inutilmente - per ogni pagina,1 useremo la sintassi liquid per farlo caricare solo quando serve.

Nei post in cui dovrai caricare Mathjax, aggiungi l’ attributo math: 'yes' nel Front Matter.

Questo attributo servirÃ  per inserire un blocco if else con liquid nel file _/includes/head.html ( o nel tag head del file html di default per le tue pagine ) per caricare dinamicamente lo script di Mathjax.
